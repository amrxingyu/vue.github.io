# 投资与投资主体

投资的定义：为了（可能不确定的）将来的消费（价值）而牺牲现在一定的消费（价值）

投资的特性：延迟消费、期望获得一定的收益、不确定性和风险

投资学内容：市场和工具介绍、资产定价、资产组合管理。

投资的主体：家庭（居民）、企业、政府。

  • 家庭收入

    • 投资：直接投资、间接投资
    • 消费：购买商品或服务

  • 企业收入

    • 投资：研发投资，员工培训、建造厂房，购买设备、存货投资
    • 消费：工资福利

  • **政府收入：**税收、政府融资

    • 政府投资：直接、间接
    • 支付支出：医疗等社会保障、公务员支出

# 单元投资与宏观经济运行的内部逻辑

  • 虚拟经济：
    • 家庭将资金配置金融资产
    • 企业将融资等所得资金配置金融资产
    • 资金闭环但没有创造任何商品或服务

question：

  • 虚拟经济的繁荣是真正的、可持续的繁荣吗？

    • 金融泡沫会导致实体经济的萎缩，并可能最终造成经济的全面衰退

  • 是否应当限制虚拟经济的繁荣？

    • 虚拟经济的适度发展对实体经济具有辅助和推动作用

  • 托宾Q系数：我国资本市场不大成熟，对此系数的使用有限制
